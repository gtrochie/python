# Tic-Tac-Toe Winner Checker ⭕❌

A Python utility that analyzes a 3x3 game board to identify the winner. This project demonstrates array indexing, matrix traversal, and input validation.

## How It Works
The program takes a nested list (matrix) as input and checks for three matching symbols in:
* **Rows**
* **Columns**
* **Diagonals**
