
# Rock Paper Scissors

## Overview
Rock Paper Scissors is a classic hand game implemented in `game.py`. Players choose rock, paper, or scissors, and the game determines the winner based on standard rules:
- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock

## How to Run the Game
```bash
python3 game.py
```

Follow the on-screen prompts to enter your choice (rock, paper, or scissors) and play against the computer.

## Running Tests
Execute the test suite with:
```bash
pytest
```

Or for verbose output:
```bash
pytest -v
```

## Game Features
- Play against a computer opponent with random moves
- Best-of-N game modes
- Input validation and error handling
